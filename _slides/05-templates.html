Slide Show (S9) Template Pack Options

What’s S5?

Simple Standards-based Slide Show System (S5) is a classic public domain (free, open source) slide show package inspired by Opera Show and others that works in all modern browsers (without any plugin required because it includes its own slide show machinery in JavaScript).

(Use the s5blank or s5themes template pack to create S5 slide shows.)

What’s S6?

S6 started as a rewrite of S5 using the jQuery JavaScript library — offering easier to understand and easier to extend code. Add plugins, effects and more. Contributions welcome!

(Use the s6blank or s6syntax template pack to create S6 slide shows.)
